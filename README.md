## Project Overview
This project is an interactive data visualization platform built with Python and Streamlit. We provide a deep dive into reported crime incidents in Chicago for the year 2026, combining public safety records with community socioeconomic indicators (poverty rates) to explore the spatial, temporal, and social dynamics of crime in the city.

The data is sourced directly from the [Chicago Data Portal](https://data.cityofchicago.org/), covering crime types, precise locations, timestamps, and police districts.

## Technical Stack
* **Streamlit**: Web framework for the interactive UI.
* **Altair**: Declarative visualization library for complex, linked charts.
* **Pandas**: Data manipulation and pre-aggregation for high-performance rendering.
* **Git LFS**: Managed large CSV datasets (2026 full crime records).
